# 文章结构定义

## 1. 经典五步法 (Classic 5-Step) - [默认]
1. **情绪/需求开场**：针对具体痛点（养老、子女教育、财富传承）切入。
2. **具体案例建模**：提供具体的“画像+金额+时间”（例如：40岁女性，年交12万，交5年）。
3. **利益点深度拆解**：区分保证与非保证，使用颜色或线条作为记忆锚点。
4. **公司底层实力分析**：谈投资策略（债券占比、股东背景、监管评级），赋予公司性格标签（如“稳健百年老店”）。
5. **总结与选购建议**：强调“最合适”而非“最好”，引导根据偏好做决定。

## 2. 视频节奏建议
### 1分钟视频：
- 15秒：提出问题
- 30秒：核心拆解/建模
- 15秒：总结收束

### 3分钟或以上视频：
- 30秒：提出问题
- 1分30秒或以上：深度拆解/建模
- 1分钟：通俗翻译 + 总结
