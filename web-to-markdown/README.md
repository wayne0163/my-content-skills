# Web To Markdown Skill

## 简介
这是一个强大的网页内容提取与转换工具。它可以接收任意网址（URL），自动抓取其中的核心内容，并将其转换为格式整洁的标准 Markdown 文档。特别优化了对微信公众号等常见中文内容平台的解析。

## 主要功能
1.  **智能抓取**：支持提取微信公众号、博客、新闻网站等页面的正文。
2.  **元数据提取**：自动识别文章标题、作者、来源平台及原始链接。
3.  **核心观点总结**：AI 自动阅读全文，提炼出 3-5 个核心观点（Bullet points）。
4.  **智能标签生成**：根据文章内容自动生成 2-5 个符合 Obsidian 格式的标签（如 `#人工智能` `#笔记`）。
5.  **格式标准化**：输出整洁的 Markdown 格式，包含专门预留的“我的心得”区域，方便后续笔记整理。

## 使用场景
*   **稍后阅读**：将想看的网页快速转为本地 Markdown 文件，存入 Obsidian 或 Notion。
*   **知识库构建**：快速收集行业文章，自动打标签归档。
*   **内容摘要**：没时间读长文时，让它先生成“核心观点”概览。

## 使用方法
在对话框中直接输入指令，附上链接即可。

**示例指令：**
> "用 web-to-markdown 把这个链接转成笔记：https://mp.weixin.qq.com/s/example..."
> "帮我把这篇文章抓下来，生成 markdown：[网址]"

## 输出示例
```markdown
# [文章标题]

- **Author**: [作者名]
- **Source**: [来源/平台] ([原始链接])
- **Date**: [日期]
- **Tags**: #关键词1 #关键词2

## Core Ideas
- [核心观点1]
- [核心观点2]
...

## Content
[正文内容摘要或全文...]

## 我的心得
[此处留空供填写]
```
