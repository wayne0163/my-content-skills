# Skills Overview

This directory contains custom skills designed for the Gemini CLI agent.

## Available Skills

### 1. [Web To Markdown](./web-to-markdown/README.md)
**Function:** Converts web pages (URLs) into standardized Markdown documents.
**Key Features:**
- Extracts metadata (Author, Source).
- Summarizes Core Ideas.
- Generates Obsidian-style tags.
- Optimized for WeChat Official Accounts.

### 2. [Personal Style Rewriter](./personal-style-rewriter/README.md)
**Function:** Rewrites content to match a specific personal writing style.
**Key Features:**
- Analyzes writing samples for "Voice" and "Tone".
- Transforms bullet points into full articles.
- Supports a "Default Persona" via the `references/` directory.

## How to Use
See the linked README files above for detailed usage instructions for each skill.
