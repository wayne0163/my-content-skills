# News Content Factory

新闻内容工厂：从搜索到成文的完整工作流。

## 一句话描述

"今日新闻" → 8步流程 → 完整文章。

## 8步工作流

1. **领域确认** - 选择关注领域和地域范围
2. **新闻搜索** - 自动搜索当日热点新闻
3. **候选展示** - 格式化展示5-10条新闻供选择
4. **组合选择** - 选择2条新闻，分析其关联性
5. **角度生成** - 提供3个不同解读角度（宏观/微观/批判）
6. **方向确认** - 选择角度，确认标题和核心论点
7. **风格/结构配置** - 选择或自定义写作风格和文章结构
8. **文章生成** - 生成完整文章，支持迭代优化

## 核心特色

- **单会话完成**：利用 Gemini CLI 长上下文，无需切换工具
- **渐进式控制**：可以"一句话搞定"，也可以每一步精细调整
- **风格记忆**：保存个人风格模板，越用越顺手
- **质量检查**：自动生成前自检，确保事实和逻辑准确

## 使用示例

### 完整流程（精细控制）

```bash
> 今日新闻
[选择领域：科技]
[展示新闻...]
[选择：2+5]
[选择角度：3-批判型]
[选择风格：A5-犀利评论]
[选择结构：2-对比冲突]
[生成文章...]
[调整结尾...]
[满意，保存风格]
```

### 快速启动（一句话）

```bash
> 用轻松解读风写今日AI新闻，对比两条，选故事角度
[直接生成]
```

## 快捷指令

| 指令 | 作用 |
|------|------|
| `/reload` | 重新读取配置 |
| `/style analyze` | 分析文本风格 |
| `/title` | 重新生成标题 |
| `/style` | 切换写作风格 |
| `/structure` | 切换文章结构 |
| `/save [name]` | 保存当前配置为快捷方案 |
| `/extend [段落]` | 扩展某段内容 |
| `/shorten [段落]` | 压缩某段内容 |
| `/fact [查询]` | 核实事实 |

## 输出质量

生成文章包含：
- ✅ 准确的新闻事实引用
- ✅ 清晰的两条新闻关联逻辑
- ✅ 独特的解读角度和洞察
- ✅ 符合指定风格的语言表达
- ✅ 适合目标平台的段落结构

## 注意事项

- 首次使用建议走完整流程，熟悉后可用快捷方式
- 风格模板需要积累，建议每次满意后都保存
- 新闻搜索依赖 Gemini CLI 的搜索能力，如未配置可手动粘贴新闻链接
