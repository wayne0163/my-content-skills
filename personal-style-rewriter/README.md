# Personal Style Rewriter Skill

## 简介
这是一个“私人订制”的写作助手。它不是机械地扩写文字，而是像一个影子写手一样，先分析并模仿特定的写作风格（语气、句式、节奏），然后将你的碎片化想法重写成一篇完整的文章。

## 主要功能
1.  **风格指纹分析**：深入分析范文的语调（幽默/严肃）、人称（第一/第三人称）、结构习惯（开篇讲故事/开门见山）和词汇特征。
2.  **小样复刻**：将粗糙的会议记录、Bullet points 或脑暴笔记，扩展成连贯通顺的文章。
3.  **默认风格库**：支持读取 `references/` 目录下的固定范文，建立你的“默认写作人格”。
4.  **动态仿写**：也可以在对话中临时提供一段文字作为模仿对象。

## 使用场景
*   **个人博客写作**：把灵感笔记快速变成符合你个人风格的博文。
*   **统一团队文风**：让不同成员的周报或公告统一成特定的团队 Tone & Manner。
*   **社交媒体运营**：把一篇严肃的技术文档改写成活泼的小红书或 LinkedIn 风格。

## 使用方法

### 方式一：使用默认风格（推荐）
先将你的代表作（Markdown 格式）放入 `references/` 文件夹中，然后直接指令：
> "用 personal-style-rewriter 帮我把这几点写成文章：1. xxx 2. xxx..."

### 方式二：临时指定风格
在对话中直接提供范文：
> "参考这段话的风格：'[粘贴范文]'，帮我重写下面这段内容：'[你的草稿]'"

## 目录结构说明
*   `SKILL.md`: 核心逻辑定义。
*   `references/`: **[重要]** 存放你的风格范文（.md 或 .txt）。Skill 会自动读取这里的文件作为默认模仿对象。
