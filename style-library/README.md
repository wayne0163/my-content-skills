# Style Library

个人写作风格库：保存、分析、调用你的专属写作风格。

## 核心功能

- **风格分析**：粘贴范文，AI自动提取风格指纹（句式、语气、修辞等）
- **模板保存**：将喜欢的风格保存为可复用模板
- **快速切换**：写作过程中随时切换风格
- **风格对比**：对比不同风格的差异，选择最适合的

## 预设风格（5种）

| 风格 | 特点 | 适用场景 |
|------|------|---------|
| 深度分析型 | 数据驱动、长句、客观 | 财经、政策分析 |
| 轻松解读型 | 口语化、短句、幽默 | 科技动态、产品分析 |
| 个人随笔型 | 故事开场、情感共鸣 | 观点表达、成长记录 |
| 国际视野型 | 全球视角、历史纵深 | 国际政治、跨国比较 |
| 犀利评论型 | 观点鲜明、战斗感 | 热点评论、争议话题 |

## 使用方式

### 独立使用

```bash
> /style list              # 查看所有风格
> /style load 轻松解读型    # 加载风格
> /style analyze [粘贴文本] # 分析文本风格
> /style save 我的风格      # 保存当前风格
```

### 在新闻工作流中使用

在 `news-content-factory` 的 Step 7 中，输入 `/style` 即可调用本库。

## 创建自定义风格

1. 找到你喜欢的文章（自己写的或别人的）
2. 使用 `/style analyze` 粘贴分析
3. 告诉AI调整方向（"更口语化"、"减少修辞"等）
4. 保存为 `style-my-voice`
5. 下次直接调用

## 风格指纹包含

- 句式长度分布
- 语气人称偏好
- 过渡词习惯
- 修辞手法偏好
- 段落长度习惯
- 标点符号特征

越用越精准，最终形成你的"AI写作分身"。
